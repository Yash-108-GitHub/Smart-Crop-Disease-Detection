## Description
A machine learning-based system that detects crop diseases from plant leaf images. The system analyzes uploaded images and predicts possible diseases to help farmers take early preventive action.

## Features
- Image-based crop disease detection
- Automatic disease classification
- Early diagnosis to reduce crop loss
- Simple interface for uploading leaf images

## Technologies Used
- Python
- Machine Learning
- Image Processing
- HTML
- CSS
- JavaScript
